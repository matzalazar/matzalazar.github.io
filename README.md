# Mi Sitio Web Personal

Repositorio de mi sitio web personal (matzalazar.github.io).

Este sitio es un portafolio estático construido con Jekyll. Su contenido se actualiza automáticamente mediante un orquestador de Python que recolecta datos de mis perfiles públicos (Goodreads, Coursera, GitHub) y los inyecta en el directorio _data. Podés revisar el proyecto recolector de datos [acá](https://github.com/matzalazar/personal-tracker).

## Créditos del Theme

La base de este sitio es `jekyll-theme-console`, un theme minimalista creado por [b2a3e8](https://github.com/b2a3e8), bajo licencia [MIT](https://github.com/b2a3e8/jekyll-theme-console/blob/master/LICENSE.txt).

He modificado el theme original para integrarlo con mi sistema de datos y adaptarlo a mi estética personal.
